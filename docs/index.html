Mijeo Studios Privacy Policies

We completely respects the user's privacy and do not collect user data that is directly linkable to the user. Note that our app uses Google Analytics to help analyse how users use the app so that we may improve the app. The information collected is not personaly identifiable to users and neither we nor Google will associate your IP address with any data. Any data that is collected is done so anonymously.

Except as outlined in the section below we do not allow any third party to use the statistical analytics tool to track or to collect any Personally Identifiable Information of users.
